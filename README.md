# java-web-bloop-sbt-spring-radis-atomic-click-me

## Description
Compare traditional static int vs.
redis atomic int.

Compiled and ran from build server `bloop`.

# Build note
Dependencies must be compatable with jdk8 or less.

## Tech stack
- bloop
- java
- bloop-sbt

## Docker stack
- openjdk:11-jdk

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
